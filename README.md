# 💍 Motor do Ateliê 3.0

Sistema de gestão especializado para ateliês de noivas, focado em controle de estoque de insumos, produção sob medida e financeiro integrado.

## 🚀 Status do Projeto

### Sprint 1: Fundação & CRM (Concluído)
- [x] Cadastro de Noivas e Fornecedores.
- [x] Agenda Inteligente (Sincronização de Provas e Casamento).

### Sprint 2: Estoque e Produção (Concluído)
- [x] Entrada de Notas com seletor de Unidade/Metragem.
- [x] Módulo de Confecção com cálculo de Markup e Mão de Obra.
- [x] **Fix:** Baixa automática de insumos no estoque após produção.
- [x] **Fix:** Estorno financeiro automático ao deletar compras/produtos.

### Sprint 3: Vendas e Pedidos Sob Medida (PRÓXIMA)
- [ ] **Módulo de Vendas:** Integração da calculadora de confecção dentro do checkout de venda.
- [ ] **Fluxo de Receita:** Geração automática de contas a receber (Entrada + Parcelas).
- [ ] **Baixa Automática:** Abatimento de materiais em tempo real para pedidos personalizados.
- [ ] **Status de Pedido:** Acompanhamento da peça (Em Produção -> Pronto -> Entregue).

## 🛠️ Tecnologias e Arquitetura
- **Backend:** Node.js + TypeScript (Prisma/Nest pronto para escala).
- **Frontend:** React + Tailwind CSS.
- **Banco de Dados:** Atualmente em memória para agilidade (MVP).
- **Controle de Versão:** Git + GitHub.
